## 🎯 Three Simple Steps to Go Live

### 1. Get OpenAI API Key (5 min)
```
Visit: https://platform.openai.com/api-keys
Create: New secret key
Copy: sk-proj-...
Save: For later
```
*Optional: App works without it, just without AI features*

### 2. Deploy Backend to Render (10 min)
```
1. Go: https://dashboard.render.com
2. New → Web Service
3. Connect GitHub: resume-analyzer
4. Fill settings from DEPLOY_NOW.md
5. Add environment variables
6. Deploy (wait 3-5 min)
→ Note your backend URL
```

### 3. Deploy Frontend to Vercel (10 min)
```
1. Go: https://vercel.com
2. Add New → Project
3. Select GitHub: resume-analyzer
4. Root directory: frontend
5. Add NEXT_PUBLIC_API_URL environment variable
6. Deploy (wait 2-3 min)
→ Get your live URL
```

**Total Time: ~30 minutes**
